# Asset Management System

## Overview

Asset Management System is a blockchain-based application developed for secure, transparent, and traceable management of institutional assets. The system enables asset registration, assignment, transfer requests, and approval processes through smart contracts.

## Features

* Asset registration
* Personnel assignment
* Transfer request creation
* Transfer approval process
* Blockchain transaction tracking
* Transparent and immutable records

## Technologies Used

* Solidity
* React
* Vite
* Ethers.js
* MetaMask
* Polygon Amoy Testnet

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the frontend directory:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Smart Contract

Contract source code:

```text
contracts/AssetManagement.sol
```

Contract ABI:

```text
contracts/AssetManagementABI.json
```

## How It Works

1. Users connect their MetaMask wallet.
2. Assets can be registered on the blockchain.
3. Assets can be assigned to personnel.
4. Transfer requests can be created for assets.
5. Authorized users can approve transfer requests.
6. All transactions are permanently recorded on the blockchain.
